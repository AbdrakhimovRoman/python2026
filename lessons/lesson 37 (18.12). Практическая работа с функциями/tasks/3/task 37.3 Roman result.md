### 1. Формулировка задания
Требуется реализовать две функции:
- `push_score(score, scores=None)`: добавляет переданную оценку `score` в список `scores` (если список не передан, создаёт новый). Возвращает обновлённый список.
- `top_scores(scores, n=3)`: возвращает `n` наибольших значений из списка `scores` (по умолчанию `n=3`).

**Ключевые требования:**
- Проверить три сценария для `push_score`:
  1. Два последовательных вызова без передачи `scores`.
  2. Вызов с передачей существующего списка.
- Для `top_scores` обеспечить значение по умолчанию `n=3`.
