### Задание 31.1. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая нормализует название учебной группы.

Требования:
- убрать лишние пробелы по краям
- заменить несколько пробелов внутри строки на один
- привести буквы к верхнему регистру
- вывести количество символов и слов в результате

### Пример входных данных
```
  python   2026  группа 
```

### Пример выходных данных
```
Название: PYTHON 2026 ГРУППА
Символов: 18
Слов: 3
```

### Критерии успешного выполнения
- строка очищена от лишних пробелов
- все буквы приведены к верхнему регистру
- длина и количество слов посчитаны корректно

### Подсказки
- используй `strip`, `split`, `' '.join(...)`, `upper()`

### Частые ошибки и ограничения
- не модифицируй строку по индексу . работай с новыми строками
- учитывай пустой ввод
