### Задание 31.3. Основы Python. Работа со строками и списками

### Описание
Напиши программу, которая помогает формировать список тренировок.

Требования:
- прочитать строку с упражнениями, разделенными `;`
- разбить строку на элементы, удалить пробелы по краям
- удалить пустые элементы
- отсортировать список упражнений по алфавиту и вывести с нумерацией

### Пример входных данных
```
  бег ; отжимания ; ; подтягивания ;  пресс  
```

### Пример выходных данных
```
1. бег
2. отжимания
3. подтягивания
4. пресс
```

### Критерии успешного выполнения
- строка корректно разбита по `;`
- пустые элементы удалены
- итоговый список отсортирован и выведен с нумерацией

### Подсказки
- используй `split(';')`, `strip()`, генератор списка

### Частые ошибки и ограничения
- не забывай проверять элементы на пустоту после `strip`
- не используй внешние библиотеки
