### Блок 1: Работа со списками

Ты ведешь список задач по подготовке к путешествию. Создай список `tasks` с тремя задачами. Добавь две новые задачи в конец списка. Выведи общее количество задач, вторую и последнюю задачу. Затем выведи нумерованный список всех задач.

Пример входных данных:
```python
tasks = ["купить билеты", "забронировать отель", "собрать вещи"]
new_tasks = ["обменять валюту", "купить сим-карту"]
```

### Блок 2: Срезы (особое внимание)

У тебя есть список городов в маршруте. 
1. Выведи первые четыре города с подписью "Начало маршрута".
2. Выведи последние три города с подписью "Конец маршрута".
3. Выведи срез с третьего по седьмой город включительно с подписью "Основная часть".
4. Выведи список без первого и последнего города с подписью "Без крайних городов".

Пример входных данных:
```python
cities = ["Москва", "Санкт-Петербург", "Казань", "Екатеринбург", "Новосибирск", "Красноярск", "Владивосток"]
```

### Блок 3: Мутация списков

У тебя есть список покупок для путешествия. 
1. Добавь в конец списка покупки: "адаптер" и "зарядка".
2. Вставь в середину списка (после второго элемента) покупку "карта".
3. Удали из списка покупку "зонт" (если она есть).
4. Перемести последнюю покупку в начало списка.

Пример входных данных:
```python
purchases = ["паспорт", "деньги", "фотоаппарат", "зонт"]
```

### Блок 4: Вложенные структуры

У тебя есть список достопримечательностей, каждая представлена словарем с полями: `название`, `город`, `время_посещения` (в часах). 
1. Распечатай информацию в формате: `Название: город (время_посещения ч)`.
2. Сформируй список названий достопримечательностей с временем посещения меньше 2 часов.
3. Создай новый список только для определенного города (выбери город из списка).
4. Подсчитай общее время, которое потребуется на посещение всех достопримечательностей.

Пример входных данных:
```python
attractions = [
    {"название": "Кремль", "город": "Москва", "время_посещения": 3},
    {"название": "Эрмитаж", "город": "Санкт-Петербург", "время_посещения": 4},
    {"название": "Мечеть Кул-Шариф", "город": "Казань", "время_посещения": 1.5},
    {"название": "Площадь 1905 года", "город": "Екатеринбург", "время_посещения": 1}
]
```
