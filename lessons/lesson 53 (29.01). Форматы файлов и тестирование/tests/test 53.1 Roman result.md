1) ✅ csv.writerows()
2) csv.read()
3) csv.dump()
  
### ИИ анализ
Метод `writerows()` объекта writer из модуля `csv` предназначен для записи списка строк (рядов данных) в CSV-файл за одну операцию. `csv.read()` не существует, а `csv.dump()` — это метод для работы с JSON, а не CSV. Ваш ответ верен, вы правильно указали эффективный метод для пакетной записи.
