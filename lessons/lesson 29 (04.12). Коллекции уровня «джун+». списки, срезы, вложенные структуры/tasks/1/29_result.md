6. Вариант полного решения
Решение практически совпадает с кодом студента:
```python
base_tasks = ["проверить код", "написать README", "упаковать архив"]
urgent = ["записать демо", "отправить письмо"]

base_tasks.extend(urgent)

total_tasks = len(base_tasks)
second_task = base_tasks[1]
last_task = base_tasks[-1]

print(f"Всего дел: {total_tasks}")
print(f"Второе дело: {second_task}")
print(f"Последнее дело: {last_task}")

for i, task in enumerate(base_tasks, start=1):
    print(f"{i}. {task}")
```
