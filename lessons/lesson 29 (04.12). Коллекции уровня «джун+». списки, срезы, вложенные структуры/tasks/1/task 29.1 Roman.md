# Задание 29.1. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Знакомство со списками. Учимся создавать списки, обращаться к элементам по индексу и аккуратно перебирать их циклом.

## Условие задачи
Собери список обязательных дел перед сдачей курсового проекта. Создай стартовый список, добавь срочные дела в конец, подсчитай общее количество, выведи второй и последний элементы. Затем покажи план с нумерацией, чтобы ничего не пропустить.

## Пример входных данных
```python
base_tasks = ["проверить код", "написать README", "упаковать архив"]
urgent = ["записать демо", "отправить письмо"]
```

## Пример выходных данных
```
Всего дел: 5
Второе дело: написать README
Последнее дело: отправить письмо
1. проверить код
2. написать README
3. упаковать архив
4. записать демо
5. отправить письмо
```

## Критерии успешного выполнения
- Список создан в коде и расширен без ручного копирования
- Используются индексы для выборки элементов
- Количество элементов считается через `len`
- Вывод с нумерацией отражает итоговый порядок дел

## Подсказки
- `extend` быстро добавит несколько срочных дел
- Индекс `1` отдаст второй элемент, `-1` — последний
- `enumerate` поможет подписать пункты числами

## Частые ошибки и ограничения
- Не обращайся к индексам раньше, чем убедишься, что список достаточно длинный
- Избегай дублирования дел при расширении
- Старайся не смешивать типы данных, элементы должны быть строками
- Проверяй, что все выводы идут после добавления новых задач
