# Задание 29.3. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Мутация и методы списков. Практикуем добавление, перестановку и удаление элементов так, чтобы не потерять исходные данные.

## Условие задачи
Ты координируешь выездную съемку и настраиваешь список оборудования:
1. Создай список базовых предметов (камера, объектив, штатив и т.д.).
2. Добавь через `extend` элементы, которые привезет ассистент.
3. Вставь «контрольный чек-лист» в позицию №2, чтобы он был сразу после камеры.
4. Удали по значению предмет, который заменили новым.
5. Перемести предпоследний элемент на первое место, используя `pop(-2)` и `insert(0, ...)`.
6. Выведи итоговый список с комментариями, зачем нужен каждый пункт.

## Пример входных данных
```python
gear = ["камера", "основной объектив", "штатив", "карты памяти"]
assistant_items = ["подвес", "запасная батарея"]
```

## Пример выходных данных
```
1. контрольный чек-лист — последовательность сборки
2. камера — основной корпус
3. основной объектив — стандартные планы
4. штатив — стабильные кадры
5. запасная батарея — подстраховка
6. карты памяти — хранение материала
```

## Критерии успешного выполнения
- Использованы минимум три разных метода изменения списка
- Вставка и удаление корректно учитывают индексы и значения
- Перестановка выполнена методами `pop` и `insert`
- Итоговый список выводится после всех операций

## Подсказки
- `extend` добавит ассистента сразу целиком
- Для вставки на вторую позицию используй индекс `1`
- Перед `remove` убедись, что элемент точно присутствует
- Нумерацию удобно делать через `enumerate(..., start=1)`

## Частые ошибки и ограничения
- Не модифицируй список в том же цикле, где его перебираешь
- Следи, чтобы индексы не выходили за длину списка
- Не обменивай элементы вручную. используй методы списка
- После завершения всех операций обязательно распечатай результат
