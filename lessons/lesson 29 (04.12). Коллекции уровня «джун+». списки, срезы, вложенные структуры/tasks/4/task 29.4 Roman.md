# Задание 29.4. Коллекции уровня «джун+». списки, срезы, вложенные структуры

## Блок урока
Вложенные структуры. Практикуем хранение данных в списках словарей и извлечение информации по нескольким признакам.

## Условие задачи
Ты ведешь список съемочных локаций и доступных таймслотов:
1. Создай список словарей `locations` с ключами `name`, `address`, `slots` (список словарей с `time` и `scene`), `indoor`.
2. Выведи все локации с адресом и отметкой, помещение это или улица.
3. Сформируй общий список всех сцен, проходящих внутри помещений (`indoor == True`).
4. Подготовь словарь вида `scene -> список таймслотов`, пройдя по вложенным структурам.
5. Подсчитай количество уличных сцен и выведи итог.

## Пример входных данных
```python
locations = [
    {
        "name": "Павильон А",
        "address": "ул. Светлая, 5",
        "indoor": True,
        "slots": [
            {"time": "10:00", "scene": "интервью"},
            {"time": "12:00", "scene": "монолог"}
        ]
    },
    {
        "name": "Площадь",
        "address": "Центр",
        "indoor": False,
        "slots": [
            {"time": "15:00", "scene": "погоня"}
        ]
    }
]
```

## Пример выходных данных
```
Локация. Павильон А. адрес. ул. Светлая, 5. внутри
  - 10:00. интервью
  - 12:00. монолог
Локация. Площадь. адрес. Центр. улица
  - 15:00. погоня
Внутренние сцены. ['интервью', 'монолог']
Карта сцен. {'интервью': ['10:00'], 'монолог': ['12:00'], 'погоня': ['15:00']}
Уличных сцен. 1
```

## Критерии успешного выполнения
- Список `locations` содержит вложенные словари со слотами
- Вывод формируется циклом с вложенным обходом слотов
- Структуры с сценами собираются программно, без ручного дублирования
- Количество уличных сцен подсчитывается на основе данных

## Подсказки
- Проверяй поле `indoor` и конвертируй `True/False` в понятный текст
- Для словаря сцен можно использовать `defaultdict(list)` или проверять наличие ключа
- Счетчик уличных сцен увеличивай на `len(slots)` для локаций под открытым небом

## Частые ошибки и ограничения
- Не забывай, что `slots` уже список словарей. не превращай его в другой тип
- Не модифицируй исходные данные при выводе
- Избегай копирования кода для каждой локации. используй цикл
- Следи за единым форматом строки вывода
