### 6. Вариант полного решения
```python
def mean_abs_deviation(*values):
    if not values:
        raise ValueError("Невозможно рассчитать среднее без входных данных.")
    average = sum(values) / len(values)
    deviations = [round(abs(x - average), 1) for x in values]
    return tuple(deviations)

# Примеры использования:
print(mean_abs_deviation(3, 5, 7, 9))  # (3.0, 1.0, 1.0, 3.0)
existing_scores = [10, 8, 6]
print(mean_abs_deviation(*existing_scores))  # (2.0, 0.0, 2.0)
```
