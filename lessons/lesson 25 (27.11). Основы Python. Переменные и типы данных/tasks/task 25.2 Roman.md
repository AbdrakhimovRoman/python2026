Задание 2. Основы Python. Переменные и типы данных

### Описание
Создай скрипт, который принимает название курса, длительность в часах и цену одного часа. Длительность приведи к int, цену приведи к float. Выведи сводку с исходными данными и общей стоимостью курса.

### Пример входных данных
```
Backend Basic
24
750.5
```

### Пример выходных данных
```
Курс: Backend Basic (str)
Часы: 24 (int)
Цена часа: 750.5 (float)
Итого: 18012.0 руб. (float)
```

### Критерии успешного выполнения
- Значения читаются через input() в порядке: курс, часы, цена часа.
- Часы преобразуются к int без ошибок.
- Цена часа преобразуется к float.
- Итоговая стоимость рассчитывается как произведение и выводится с типом.
- Скрипт завершается без исключений при корректном вводе.

### Подсказки
- Храни результат type(...)__name__ в переменной, если используешь его несколько раз.
- Применяй f-строки, чтобы собрать вывод компактно.
- Выполняй преобразования сразу после чтения значений, чтобы не запутаться.

### Частые ошибки и ограничения
- Не меняй порядок ввода.
- Не округляй итоговую стоимость вручную.
- Не добавляй в вывод посторонние сообщения.
