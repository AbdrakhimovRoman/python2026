### Отчёт по проверке решения задания

**1. Формулировка задания**  
Цель: обработать список заказов `spots` (район, количество), накопить суммы в словаре `orders`, удалить районы из списка `finished` с выводом суммы удалённого района (или сообщения "нет заказов"), затем вывести итоговую сумму для выбранного района (например, 'center').  
Ключевые ограничения:
- Накопление значений через методы `get`/`setdefault`.
- Удаление через `pop(key, 'нет заказов')` для избежания `KeyError`.
- Вывод результата удаления и итоговой суммы после обработки.

**6. Вариант полного решения**  
```python
spots = [('center', 18), ('south', 9), ('center', 7), ('north', 4)]
finished = ['south']
orders = {}

# Накопление заказов с использованием get
for district, amount in spots:
    orders[district] = orders.get(district, 0) + amount

# Удаление районов и вывод результата
for district in finished:
    result = orders.pop(district, 'нет заказов')
    print(f'Результат удаления района {district}: {result}')

# Вывод итоговой суммы для выбранного района
selected_district = 'center'
total = orders.get(selected_district, 0)
print(f'Итоговая сумма заказов {selected_district}: {total}')
```
