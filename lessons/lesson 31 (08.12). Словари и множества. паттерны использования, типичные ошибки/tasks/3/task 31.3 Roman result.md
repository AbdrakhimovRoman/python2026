### Отчёт по проверке задания 31.3

---

#### 1. Формулировка задания
**Цель:** Найти поставщиков, участвующих одновременно в дневной (`set_a`) и вечерней (`set_b`) смене (пересечение), а также определить поставщиков, которых нужно добавить в вечернюю смену (элементы `set_b`, отсутствующие в `set_a`).  
**Ограничения:**
- Использовать операции над множествами (`&`, `-`, `|`).
- Не преобразовывать множества в списки без необходимости.
- Вывод должен содержать два множества с пояснительными подписями.

---

#### 2. Результаты проверки
**Файл:** `zadacha3`  
**Код студента:**
```python
set_a = {'ramen', 'gelato', 'coffee'}
set_b = {'ramen', 'bbq', 'tapas'}
intersection = set_a & set_b
difference = set_b - set_a
print("Пересечение:", intersection)
print("Добавить вечером:", difference)
```
**Тестирование:**
- Запущен код с примером входных данных.
- Результат:
  ```
  Пересечение: {'ramen'}
  Добавить вечером: {'bbq', 'tapas'}
  ```
  Совпадает с примером из условия.
